# data_visualize_flaskapp

로컬 데이터를 사용한 데이터 필터링/데이터 시각화 및 데시보드 웹 어플리케이션.

### Requirments
Python 3.8

### How to use
1. Clone this repository
2. Install dependencies
3. Run app.py
4. Enter http://127.0.0.1:5000/
